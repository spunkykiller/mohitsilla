Minimal, single-page portfolio website built with Next.js and Tailwind CSS.
Designed for high performance, dark mode aesthetics, and operational credibility.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Font:** Inter / IBM Plex Sans
- **Icons:** Lucide React
- **Animations:** Framer Motion
- **Deployment:** Vercel

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Build for production:**
    ```bash
    npm run build
    npm start
    ```

## Edit Instructions

- **Content:** Edit text and links in `app/page.tsx`.
- **Styling:** Configuration in `tailwind.config.ts` and global styles in `app/globals.css`.
- **Layout/SEO:** Metadata and fonts in `app/layout.tsx`.
- **Images:** Add new images to `public/images/` and reference them using `next/image`.

## Deployment

1.  Push to GitHub.
2.  Import project into Vercel.
3.  Deploy (Output settings should be automatically detected).
